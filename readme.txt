== Description ==
Customize your WooCommerce store with a click of few buttons. This plugin provides an options panel where you can add your customizations and save them without needing to write any PHP code. And, it's absolutely free.


= Plugin Features =

* Works with any WooCommerce Theme
* Over 30+ customizations
* Take care of the minor details of your WooCommerce store
* No need to write a single line of code

* **FREE Forever**

== Installation ==

To manually install this plugin, please follow the instructions below.

1. Upload `customize-woo` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to 'Customizer' option under the 'WooCommerce' menu.
4. Configure the plugin and start enjoying it.
