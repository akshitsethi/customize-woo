== Installation ==

To manually install this plugin, please follow the instructions below.

1. Upload `woo-customizer` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to 'Woo Customizer' options under the 'Settings' menu.
4. Configure the plugin and start enjoying it.
